## 🧩 תיאור הפרויקט

### תחום הבעיה

- גנומיקה (Genomics)
- ביולוגיה חישובית (Bioinformatics)
- למידה עמוקה (Deep Learning)
- ניתוח דאטה רפואי (Medical Data Analysis)

### הפתרון המוצע

- **משימה עיקרית:** סיווג רב-מחלקתי של רצפי DNA
- **קלט (Input):** רצף DNA באורך קבוע (למשל 200 בסיסים), מיוצג ב־One-Hot Encoding
- **פלט (Output):** וקטור הסתברויות (Softmax) עבור 3 קטגוריות:

  - Class 0: אזור גנומי רגיל
  - Class 1: אזור פרי-טלומרי
  - Class 2: אזור טלומרי

- **משימה משנית:** חיזוי השפעת וריאנטים באמצעות _In-Silico Mutagenesis_ על בסיס המודל המאומן.

> _In-Silico Mutagenesis_ היא שיטה חישובית המדמה את ההשפעות של מוטציות גנטיות על גנים או חלבונים, ללא צורך בניסויים מעבדתיים. בתהליך זה משנים באופן שיטתי נוקלאוטידים או חומצות אמינו ומחזים כיצד כל שינוי משפיע על מבנה, תפקוד או ביטוי. גישה זו מאפשרת חקר רחב ומהיר של מוטציות אפשריות והבנת מנגנוני מחלה.

### 🧱 מטרות ותוצרי מפתח

1. **תוצר 1 – המודל:**  
   מודל `TeloClassifier` מאומן (ב־PyTorch או TensorFlow) המסוגל לסווג רצפי DNA לשלוש קטגוריות בדיוק גבוה (Precision / Recall גבוהים).  
   הפלט: softmax לפי הסתברויות הסיווג.

2. **תוצר 2 – ניתוח מוטציות:**  
   “מפת חשיבות” המראה כיצד שינוי בסיס בודד (SNP) משפיע על ציון הסיווג.

3. **תוצר 3 – פרשנות:**  
   ויזואליזציה של הפילטרים בשכבת ה־Conv1D להוכחת למידה של מוטיבים ביולוגיים.

### 🧬 מקורות דאטה

| קטגוריה               | מקור                     | תיאור                                |
| --------------------- | ------------------------ | ------------------------------------ |
| טלומרים (Class 2)     | TelNet Database / SimTel | מאגר ידוע + סימולטור לרצפים טלומריים |
| פרי-טלומרים (Class 1) | UCSC Genome Browser      | אזורים Subtelomeric                  |
| גנום רגיל (Class 0)   | UCSC / Ensembl           | אזורים מקודדים ובין-גניים            |

### 💾 ייצוג נתונים

- **שיטה:** One-Hot Encoding
- **תהליך:** כל בסיס מיוצג כוקטור בגודל (N,4):
  - A = [1,0,0,0]
  - C = [0,1,0,0]
  - G = [0,0,1,0]
  - T = [0,0,0,1]

**קלט למודל:** טנסור בגודל (batch size, sequence length, 4)

### 🧠 ארכיטקטורת רשת הנוירונים

מודל היברידי המשלב **CNN לזיהוי תבניות מקומיות** עם **RNN ו־Attention להבנת הקשרים ארוכי טווח**.

1. Input (N,4)
2. Conv1D → ReLU → MaxPooling1D
3. BatchNormalization + Dropout
4. Bidirectional LSTM (Bi-LSTM)
5. Attention Mechanism
6. Dense Layer
7. Output Layer (Dense + Softmax)

### 📊 הערכה ואימות

- **מדדים:**

  - Accuracy
  - Precision, Recall, F1-score לכל קטגוריה
  - Confusion Matrix (בדיקת בלבול בין טלומר לפרי-טלומר)
  - AUC-ROC / PR Curve

- **מתודולוגיה:**  
  חלוקה ל־Train / Validation / Test על בסיס רצפים מהגנום האנושי.

### 🔎 פרשנות ורלוונטיות

- **ויזואליזציית פילטרים:**  
  הצגת המשקולות של שכבת ה־Conv1D כמוטיבים ביולוגיים (כגון `'TTAGGG'`).

- **Attention Maps:**  
  מיפוי חזותי של חלקי הרצף החשובים ביותר להחלטת המודל.

- **ניתוח מוטציות:**  
  גרף המציג את הירידה בציון ה־Softmax בעקבות כל שינוי בסיס לאורך רצף טלומר קנוני.
